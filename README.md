# Микропроцессоры

Добро пожаловать в репозиторий **Микропроцессоры**! Этот репозиторий содержит экзаменационные работы по предмету микропроцессоры.

## Содержание

### Fasm

#### инструкция по с борке файлов

```bash
fasm name.asm
ld name.o -o name
./name
```

Сложение `add`
Вычитание `sub`
Умножение `imul`
Деление `idiv`

### KP
файлы `KP580`
